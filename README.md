# 🌸 Yuri点点的魔法相册

一个可可爱爱的照片管理网站，专为Yuri点点设计！

✨ **在线访问**: https://yuri-photo-gallery.vercel.app

## 🎀 功能特点

- 📸 **照片上传** - 支持拖拽上传，简单快捷
- 🗂️ **智能分类** - 家人/旅行/美食/萌宠/日常五大分类
- 🌸 **瀑布流画廊** - 美观的照片展示，支持网格/瀑布流切换
- 💫 **查看器** - 点击照片放大查看，支持键盘左右切换
- 🎨 **可爱UI** - 粉色系Kawaii风格，满满的少女心
- 📱 **响应式** - 手机、平板、电脑都能完美使用
- 💾 **本地存储** - 照片存在浏览器里，不用担心丢失

## 🛠️ 技术栈

- HTML5 + CSS3 + JavaScript (纯前端)
- LocalStorage 数据存储
- Vercel 部署
- 阳仔的爱心 💕

## 📝 使用说明

1. 点击"上传照片"选择图片
2. 填写标题和分类
3. 点击"上传到魔法相册"
4. 在画廊中查看和管理照片
